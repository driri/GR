# Ganit Richter — Simple Academic Website

This repository contains a very simple static academic website for GitHub Pages.

## Files

- `index.html` — the website content
- `style.css` — the website design
- `README.md` — these instructions

## How to edit the site

Open `index.html` in any text editor and edit the text inside the sections.

Useful places to edit:

- Name and title: near the top of the `hero` section
- About: inside the section marked `ABOUT`
- Research interests: inside the section marked `RESEARCH INTERESTS`
- Current projects: inside the section marked `CURRENT PROJECTS`
- Publications: inside the section marked `SELECTED PUBLICATIONS`
- Teaching: inside the section marked `TEACHING`
- Talks and conferences: inside the section marked `TALKS / CONFERENCES`
- Contact links: inside the section marked `CONTACT`

To change colors or spacing, edit `style.css`.

## How to publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload these three files to the repository:
   - `index.html`
   - `style.css`
   - `README.md`
3. Go to the repository settings.
4. Open **Pages**.
5. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
6. Click **Save**.
7. GitHub will publish the site and show a link.

The site should usually appear at:

`https://YOUR-USERNAME.github.io/YOUR-REPOSITORY-NAME/`

If the repository is named exactly `YOUR-USERNAME.github.io`, the site will appear at:

`https://YOUR-USERNAME.github.io/`

## Notes

- This site uses only HTML and CSS.
- There is no JavaScript.
- There is no backend.
- It is suitable for GitHub Pages.
